HTTP-Performance-Analyzer by camposr

A script that reads web server log file and generates some perfomance metrics and charts

Dependencies

Perl Chart::Clicker

Install

Make sure that you have modified your http server log file so that the last field is the time the server took to serve the request (%D if you're using Apache/mod_log_config, $request_time if you're using nginx). Usually you'll get better results if you have at least a whole week worth of data, but your mileage may vary if your server handles large traffic.
